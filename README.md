This repository contains the material relative to the work presented in "Stratified Multilevel Graphical Models: Examining Gender Dynamics in Education". 

In particular, the following files are stored: 
1. The original OECD-PISA dataset 2018 for Italy - this is obtained by filtering the observations relative to Italy from the OECD_PISA 2018 dataset in teh website https://www.oecd.org/en/data/datasets/pisa-2018-database.html
2. The pre-processing R script to obtain the clean and final dataset data_gen.csv
3. data_gen.csv
